Calculadora de Bhaskara

Conceitos Preliminares

A Fórmula de Bhaskara calcula as raizes de equações do segundo grau, isto é, equações em que a maior potência é 2. Abaixo, está uma calculadora para encontrar as raízes dessas equações. Caro usuário, para utilizar lembre-se que essas equações são do tipo:

\[ax^{2}+bx+c=0,\]

onde os coeficientes a, b e c são números reais, mas a ≠ 0.

Observações

Note que o botão CALCULAR muda de cor, no caso do click, para diferenciar as três possíveis situações para as soluções de equações do segundo grau. São elas:

  • Verde: Delta maior que zero implicando em duas raízes;
  • Violeta: Delta igual a zero;
  • Vermelho: Delta menor que zero;

Equações

As equações utilizadas nesse projeto são:

  • \( \Delta=b^{2}-4ac \)     (Para calcular o delta)
  • \(x_{1,2}=\frac{-b \pm \sqrt{\Delta}} {2a}\)     (Para calcular as raízes)
